Overview

Remote-controlled media player for a TV connected to a PC. There are cases where a person plays a media downloaded in his laptop connecting the laptop to a TV via an HDMI cable. With this application, the user can manipulate the media playing by an additional device like smartphone from his chair through HTTP connection.
